## âš™ï¸ Installation

### Prerequisites
- **Node.js**: Ensure Node.js is installed on your system.

### Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/boukarman/Blogify.git
   cd blogify
   ```

2. **Backend Setup**:
   ```bash
   cd server
   npm install
   npm run dev
   ```

3. **Frontend Setup**:
   ```bash
   cd client
   npm install
   npm run dev
   ```
